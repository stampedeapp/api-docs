# Price

A object representing the financial price for an item.

```json
{
  "value": 5000,
  "currencyCode": "GBP"
},
```

### Properties

<Property
  name='value'
  type='number'
>
  The value in the currencies *minor unit*. For example, pence for GBP Sterling.
</Property>

<Property
  name='currencyCode'
  type='string'
>
  The current code for the price. Uses [ISO 4217](https://en.wikipedia.org/wiki/ISO_4217) standard. For example, `GBP`. 
</Property>
