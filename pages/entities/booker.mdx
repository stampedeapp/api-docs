# Booker

Contact information for the person who made a booking.

```json
{
  "firstName": "Jean-Luc",
  "lastName": "Picard",
  "email": "jean@earlgrey.com"
},
```

### Properties

<Property name="firstName" type="string">
  The first name of the booker.
</Property>

<Property name="lastName" type="string">
  The last name of the booker.
</Property>

<Property name="email" type="string">
  The email address of the booker.
</Property>
